# ai_portfolio_streamlit_deployment
Transformer le modèle final_titanic_classifier.pkl en une application web.

A. Objectif Principal
Déploiement : Transformer le modèle final_titanic_classifier.pkl en une application web conviviale.

Compétence Démontrée : Maîtrise des outils MLOps de base, compréhension du cycle de vie complet du ML (de l'entraînement à l'application).

B. Outils Nécessaires
Modèle Entraîné : Le fichier final_titanic_classifier.pkl.

Streamlit : Une bibliothèque Python pour créer des interfaces web de données/ML.

Joblib/Pickle : Pour charger le modèle.
